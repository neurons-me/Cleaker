 Lógica pura
En lógica clásica solo existen:
0 / 1
falso / verdadero
sí / no
Todo sistema computable se construye sobre eso.
Y todo lo complejo sería simplemente una iteración fractal de esa bifurcación básica.
La estructura mínima necesaria para que exista cualquier sistema.
“Si puedes computarlo, no puedes perderlo.”
Porque computar implica bifurcación.
Si el universo es “plano” en ese sentido, entonces:
La jerarquía no es fundamental, es emergente.
Solo hay combinaciones.
Todo es geometría emergente de una decisión binaria repetida infinitamente.
 Primitivas Lógicas:
No es “verdadero / falso”.
Es algo anterior:
• ocupar / no ocupar
• movimiento / no movimiento
• presencia / ausencia
Eso es estructura mínima de existencia.
Nivel 0: Distinción
Para que exista cualquier sistema necesitas una sola cosa:
Que algo sea distinguible de lo que no es.
Eso ya implica:
A / no-A
Pero en esta versión geométrica lo estamos bajando todavía más:
Un punto puede estar activo o inactivo.
Eso es literalmente el bit.
No el bit como tecnología, sino como evento ontológico mínimo.
 Geométricamente
Imagina el espacio más simple posible:
Un punto.
Ese punto tiene dos estados:
0 → vacío
1 → ocupado
No necesitas dimensiones todavía.
No necesitas tiempo.
No necesitas energía.
Solo necesitas:
posibilidad de estado.
Eso ya es suficiente para que exista información.
Movimiento / no movimiento

Estado cambia → hay movimiento
Estado no cambia → no hay movimiento
Pero:
Movimiento es simplemente:
cambio de ocupación entre estados.

A → no-A
o
no-A → A
