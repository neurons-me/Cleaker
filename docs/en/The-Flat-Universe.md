# **The Flat Universe**
Reality has one primitive: **distinction.**  
`0 / 1`  
`A / not-A`  
`occupy / not occupy`

**Everything computable is built from that.**  
Hierarchy is not fundamental. It is emergent.  
Dimensions, geometry, complexity, consciousness — all are iterations of a binary decision.

**Movement is change of occupation.**  
`A → not-A`

The base needs no purpose. No intention. No direction.  
Only one rule: **allow distinction.**

> **If you can compute it, you cannot lose it.**  
> **Everything that is possible, is allowed.**

`distinction → information → geometry → landscape`
